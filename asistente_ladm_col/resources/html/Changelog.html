Cambios en la version 1.6.3

  • Ahora se revisa la versión de QGIS y se avisa al usuario si debe actualizarla.
  • Ahora se activa el plugin Processing automáticamente cuando el usuario utilice una herramienta del Asistente LADM_COL que requiera Processing.
  • Fix en guardar parámetros de configuración.
  • Se agrega registro de cambios de versión en el diálogo Acerca de.
  • Fix en cargue automático de menús correspondientes a modelos extendidos.
  • Se agrega posibilidad de selección de objetos relacionados (ej. fuentes o terrenos) desde los wizards de crear RRR y Fuente Espacial.
